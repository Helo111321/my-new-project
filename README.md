# my-new-project
智能温室⾃动控制系统

传统温室种植模式高度依赖人工巡检，不仅监测效率有限，环境参数调节存在明显延迟，同时人力投入成本较高，难以实现精细化管控。为有效改善上述不足，本文研制并完成一套基于 STM32 单片机的智能温室自动控制系统。该系统以 STM32F103 单片机作为主控核心，遵循模块化设计理念，整体架构包含核心控制、环境信息采集、执行驱动、人机交互、电源供给以及报警防护等多个功能单元。
系统借助 DHT11 温湿度传感器、光敏传感器与土壤湿度传感器，对温室内空气温湿度、光照强度及土壤含水率等核心指标进行不间断采集，单片机在完成数据解析与逻辑判断后，可依据设定参数阈值自主控制风扇、灌溉水泵与植物补光灯等设备，实现对环境的降温、补水与补光调节。为适配不同种植场景需求，系统支持手动控制与自动控制两种工作模式，可实现平稳切换，通过 LCD 显示屏实时反馈环境数据，并搭载独立看门狗与蜂鸣器报警机制，保障系统长期稳定运行与异常情况及时预警。
在实际温室环境中开展软硬件联调与测试，结果表明该系统能够稳定完成环境参数采集、智能调控与超限报警等任务，采集数据精度较高，执行设备响应及时，整体运行可靠稳定。该系统具备造价低廉、安装部署简便、实用性突出等优势，可较好满足中小型温室的智能化种植管理要求，为现代农业朝着精准化、智能化方向升级提供了可行的技术方案与实践参考。
